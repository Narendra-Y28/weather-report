# Weather Forecast App

This is a simple React application that allows users to get weather forecasts for different locations based on latitude and longitude. The app fetches weather data from an API and displays it in a user-friendly way.

## Features

- Input latitude and longitude to get the weather forecast.
- Displays maximum and minimum temperatures, wind speed, sunrise and sunset times for each day.
- Weather data is displayed in a clean and simple card layout.
- Fetches historical weather data for a specific date range.

## Tech Stack

- React
- Axios (for API calls)
- Open-Meteo API (for weather data)

## How to Run

1. Clone this repository.
2. Install dependencies:
   ```
   npm install
   ```
3. Start the development server:
   ```
   npm run dev
   ```

Open your browser and go to `http://localhost:5173` to use the app.

Enjoy checking the weather!
